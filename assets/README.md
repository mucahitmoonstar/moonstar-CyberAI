# Screenshots

Add your MoonStar CyberAI application screenshots here.

## How to add a screenshot

1. Run the application with `npm start` or `npm run dev`
2. Take a screenshot of the full interface at `http://localhost:3000`
3. Save it as `screenshot.png` in this directory

The README.md will automatically display the screenshot from this location.

---

**Screenshot Specifications:**
- Format: PNG
- Recommended size: 1920x1080 or similar aspect ratio
- Show the code editor on the left and analysis results on the right
- Include a sample code snippet being analyzed
